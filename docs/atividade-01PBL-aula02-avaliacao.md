#Esta atividade foca em listar os problemas do Localeats (aplicação de comida ficticia) no caso o nosso PBL , e fazer a associação
destes problemas com competencias da ISO25010

| Problema identificado | Atributo de qualidade afetado | Justificativa técnica |
|----------------------|------------------------------|----------------------|
| Lentidão em horários de pico | Eficiência de Desempenho \| Comportamento Temporal | Tempo de resposta elevado quando há muitos usuários simultâneos prejudica a experiência |
| Algumas telas são confusas e pouco intuitivas | Usabilidade \| Operacionalidade / Aprendizagem | Interface difícil de entender aumenta o esforço do usuário e dificulta o uso do sistema |
| Há inconsistências entre a versão web e a versão mobile | Compatibilidade \| Interoperabilidade | As versões não compartilham corretamente os dados e comportamentos esperados |
| Certas buscas retornam resultados incorretos | Adequação Funcional \| Corretude Funcional | O sistema não entrega resultados corretos, comprometendo a funcionalidade principal |
| O aplicativo apresenta falhas em determinados smartphones | Portabilidade \| Adaptabilidade | O sistema não se adapta corretamente a diferentes dispositivos e ambientes |
| Usuários encontram dificuldade para concluir ações simples | Usabilidade \| Operacionalidade | Fluxos mal projetados dificultam a execução de tarefas básicas pelo usuário |
| Avaliações desaparecem após atualização da página | Confiabilidade \| Recuperabilidade / Maturidade | O sistema não mantém dados corretamente após atualização, indicando falha de persistência |
